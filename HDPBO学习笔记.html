3.2 生成离散化的样本空间¶

真正进行拟合的样本是[x,x']的形式，所以样本空间是二维平面的点。这里的离散化在[0,1]上均匀取1000个点作为$\mathcal{X}$，然后$\mathcal{X} \times \mathcal{X}$得到一张1000*1000的网格。

In [14]:
## uniform_grid函数：形成在一个区间上的均匀数值（目的是为了将目标函数的定义域连续区间离散化）
# 如：uniform_grid(1, 1000, 0, 1)
# 即：在一维区间[0,1]上均匀生成1000个点

def uniform_grid(input_dims, num_discrete_per_dim, low, high):
    """
    Returns an array with all possible permutations of discrete values in input_dims number of dimensions.
    :param input_dims: int
    :param num_discrete_per_dim: int
    :param low: int
    :param high: int
    :return: tensor of shape (num_discrete_per_dim ** input_dims, input_dims)
    """
    num_points = num_discrete_per_dim ** input_dims
    out = np.zeros([num_points, input_dims])
    discrete_points = np.linspace(low, high, num_discrete_per_dim)
    for i in range(num_points):
        for dim in range(input_dims):
            val = num_discrete_per_dim ** dim
            out[i, dim] = discrete_points[int((i // val) % num_discrete_per_dim)]
    return out

## 给出点，conbinations函数返回这些点构成的所有的“两两组合”
# 如：将上一步的均匀分布在[0,1]上的1000个点输入，则返回这1000个点的两两组合1000x1000

def combinations(points):
    """
    Given d-dimensional points, return all pair combinations of those points
    :param points: tensor of shape (n, d)
    :return: tensor of shape (n ** 2, d * 2)
    """
    n = points.shape[0]         # 点的数量
    d = points.shape[1]         # 点的维度

    out = np.zeros((n*n, d*2))     # 初始化输出，按照上面的说明（显然）
    for i in range(n):
        for j in range(n):
            out[i*n+j][0:d] = points[i]      # 一步步给输出赋值
            out[i*n+j][d:d*2] = points[j]
    return out

def forrester(x):
    """
    1-dimensional test function by Forrester et al. (2008)
    Defined as f(x) = (6x-2)^2 * sin(12x-4)
    :param x: tensor of shape (..., 1), x1 in [0, 1]
    :return: tensor of shape (..., )
    """
    x0 = x[..., 0]        # 取出第0列数据（基于numpy的数组）
    return (6 * x0 - 2) * (6 * x0 - 2) * np.sin(12 * x0 - 4)       # 返回函数值

4.1.3 观测过程中的噪声模型¶

在观测的时候会有噪声，即$ y = f(x) + \epsilon $，$\epsilon$为噪声，这里给出了无噪声模型以及Gumble和Gauss两种有噪声模型的函数定义，这三个函数会在后面用到。

这三个函数需要输入$f(x)$和$f(x')$对应的值（当然高维duels的也可以，即多个点比较），将这两个值作为一个数组fvals输入函数，同时输入噪声模型所需要的参数（如gauss噪声的标准差，当然模型也设定有默认值），函数将返回这个duels中更偏好的点的下标（这里是假设问题是求目标函数最大值，因此返回加噪后函数值较大的点的下标）。

无噪声模型

即直接返回查询结果中较大值所在的下标

In [22]:
## 无噪声模型
# 输入参数fvals是一维数组，大小与num_choice一致（每次两个点参与比较时，num_choice值为2）

def preference_noiseless(fvals):
    # :param fvals: 1d-array of size num_choice
    assert len(fvals.shape) == 1            # 判断输入参数fvals的维数是否为1维，否则报错（代码稳健性报错）
    return np.argmax(fvals)                 # 返回fvals中最大值所在位置的索引（没有做任何处理）

耿贝尔Gumble分布

广义极值分布：$ P(X<x) = {\rm exp}(-[1+k(\frac{x-\mu}{\sigma})]^{-\frac{1}{k}})\,, $

其中 k 为形状参数，μ 为中心参数，σ 为展宽参数。称 k → 0 的分布为耿贝尔分布，也叫做第 I 类极值分布。

In [23]:
## Gumble噪声模型
# 输入参数fvals是一维数组，大小与num_choice一致（每次两个点参与比较时，num_choice值为2）
# 输入参数indifference_threshold为无差异阈值（gumble的一个参数）

def preference_based_on_gumble(fvals, indifference_threshold=0.0):
    # :param fvals: 1d-array of size num_choice
    assert len(fvals.shape) == 1            # 判断输入参数fvals的维数是否为1维，否则报错（代码稳健性报错）

    num_choice = fvals.shape[0]

    indifference_mat = (1.0 - np.eye(num_choice)) * indifference_threshold

    base_f = fvals.reshape(-1, 1) + indifference_mat * indifference_threshold

    pref_prob = np.exp(fvals - spmc.logsumexp(base_f, axis=-2))
    indifference_prob = 1.0 - np.sum(pref_prob)

    prob = np.concatenate([pref_prob, [indifference_prob]], axis=0)
    cprob = np.cumsum(prob)

    t = np.random.rand()
    selected_idx = np.searchsorted(cprob, t)

    if selected_idx == num_choice:
        # indifferent
        selected_idx = -1

    return selected_idx

高斯Gauss分布（正态分布）

$ f(x) = \frac{{\rm exp}(-x^2/2)}{\sqrt{2\pi}} $

$ X \sim N(\mu,\sigma^2),Y=\frac{x-\mu}{\sigma} \sim N(0,1) $

In [24]:
## Gauss噪声模型
# 输入参数fvals是一维数组，长度为2（这里仅限num_choice值为2的情况，因为这个函数是基于这两个点f(x)与f(x')之差构建的）
# 输入参数noise_std为高斯分布的标准差（gauss的一个参数）

def preference_based_on_gauss(fvals, noise_std=0.1):
    # :param fvals: 1d-array of size 2
    assert len(fvals.shape) == 1 and fvals.shape[0] == 2            # 判断输入参数fvals的维数是否为1维且长度为2，否则报错（代码稳健性报错）

    z = (fvals[0] - fvals[1]) / np.sqrt(2) / noise_std     # 用两点f(x)与f(x')之差构建正态分布
    
    # 前文有import scipy.stats as spst
    prefer_zero_prob = spst.norm.cdf(z)       # 求z点的概率分布（累积概率密度）

    t = np.random.rand()                      # t是一个[0,1)上的随机数
    if t <= prefer_zero_prob:                 # 返回偏好值
        # prefer x[0]
        return 0
    else:
        # prefer x[1]
        return 1

4.2 获取偏好结果的方法¶

根据目标函数的观测情况，得出带有噪声的偏好结果。

In [25]:
## gen_observation_from_f函数：获取加噪输出
# 输入参数x：duels的数组，里面包含很多个duels
# 输入参数fx：每一对duel对应的真实的观测值（objective函数的输出值）
# 输入参数noise_model：选择的噪声模型（以及模型的有关参数）

# 这里定义的常量分别对应上文提及的三种噪声，对应于函数gen_observation_from_f的noise_model输入参数
NOISELESS = 0
GUMBLE = 1
GAUSS = 2

def gen_observation_from_f(x, fx, noise_model, noise_std=0.1, indifference_threshold=0.0):
    """
    param x: [num_data] list of 2d-array [:,d]
    param fx: [num_data] list of 1d-array [:,1]
    param noise_model: either NOISELESS or GUMBLE or GAUSS
    param noise_std: standard deviation of Gaussian noise (only for Gauss noise model)
    param indifference_threshold: threshold for indifference used for Gumble noise only
    """
    
    # 对应的三种噪声模型的选取，结合前文：噪声返回的是更偏好的那个点的下标
    if noise_model == NOISELESS:
        selected_idxs = [preference_noiseless(fvals.squeeze()) for fvals in fx]      # 这里的.squeeze是基于np.array的，去除维度为0的维度（就是去掉多余的[]）

    elif noise_model == GUMBLE:
        selected_idxs = [preference_based_on_gumble(fvals.squeeze(), indifference_threshold) for fvals in fx]

    elif noise_model == GAUSS:
        selected_idxs = [preference_based_on_gauss(fvals.squeeze(), noise_std) for fvals in fx]

    else:
        raise Exception("Noise model should be either NOISELESS or GUMBLE or GAUSS")
    
    # 通过上面的步骤，已经将对应的噪声模型给出的更偏好的下标形成数组存在selected_idxs中
    
    # prefs数组用来存放每对duels中最偏好的那个点的坐标（仅x，不含观测值）
    prefs = []
    for i, selected_idx in enumerate(selected_idxs):

        if selected_idx >= 0:           # 是正确的下标值
            prefs.append(x[i][selected_idx, :].reshape(1, -1))     # 选出x中第i对duel中根据偏好选中的x，存入prefs数组中；.reshape(1,-1)是将数据（该点的坐标）转化成一行

        else:  # indifferent
            prefs.append(None)          # 这里相当于直接丢弃了这一组数据，因为i也是递增的

    return prefs

def get_noisy_observation_dts(X, objective):
    """
    :param X: tensor of shape (num_data, input_dims * 2)
    :param objective: objective function
    """
    
    # X中有多少组数据（即返回第0维的元素个数）
    num_data = X.shape[0] 
    
    # 通过std_representation()函数得到输入数据的标准表示形式（上文），便于数据处理
    X_std = std_representation(X, num_choices) 
    
    # 获取直接观测函数得到的观测值，即基于x得到的fx
    f = objective_get_f_neg(X_std, objective)
    
    # 通过gen_observation_from_f()函数获取观测值数组，返回的是每次最偏好的那个点的坐标构成的数组，参数1表示基于Gumble噪声
    obs = np.array(gen_observation_from_f(X_std, f, 1)) 
    
    # 生成空的一维数组，数组长度为X的duels的个数
    ret_val = np.zeros((num_data, 1), dtype=np.int8)
    
    for i in range(num_data):
        if np.allclose(X_std[i, 0], obs[i, 0]):      # 如果每个对应元素都相等（其实就是高维情况下的相等比较），则记ret_val[i]为1，否则0
            ret_val[i] = 1
    return ret_val

def flip(X):
    """
    :param X: tensor of shape (num_data, input_dims * 2)
    :return: tensor of shape (num_data, input_dims * 2), where the first input_dims is swapped with the second
    """
    # x和x'换位置
    input_dims = X.shape[-1] // 2   # 获取输入维度
    ret_val = np.zeros(X.shape)     # 生成空张量
    for i in range(X.shape[0]):     # 互换位置，存储在返回值中
        ret_val[i, :input_dims] = X[i, input_dims:]
        ret_val[i, input_dims:] = X[i, :input_dims]
    return ret_val

def std_representation(X, num_choices):
    """
    :param X: tensor of shape (num_data, input_dims * num_choices)
    :return: tensor of shape (num_data, num_choices, input_dims)
    """
    
    input_dims = X.shape[-1] // num_choices                             # 计算输入点的维度
    ret_val = np.zeros((X.shape[0], num_choices, input_dims))           # 构建空数组 tensor of shape (num_data, num_choices, input_dims)

        ret_val[:, i, :] = X[:, input_dims * i:input_dims * (i + 1)]    # 填值

    return ret_val

## 对于combine kernel，即下文中提到的product形式的combine kernel，计算模型部分带来的loss

@tf.function
def lengthscale_regularizer(kernel):  # for product kernel
    loss = 0
    for k in kernel.kernels:
        # 按照下面的公式计算loss并返回
        loss += 0.5 * tf.reduce_sum(tf.square((k.lengthscale - lengthscale_mean_regularizer) / lengthscale_std_regularizer))
    return loss

## 定义要训练的GP模型
# 每调用这个函数一次，即完成一次循环，生成一对新的duel[x,x']

num_steps = 3000        # 优化的次数

def train_and_visualize(X, y, lengthscale, title, num_steps=num_steps):
    
    ## 定义GP模型
    
    # 核函数kernel的选择
    # 这里使用gpflow.kernels.RBF即径向基（指数平方）核函数：k(r) = σ² exp{-½ r²}
    # 关于active_dims参数，参见gpflow.kernels.Stationary方法
    # r是输入点间的Euclidean distance，由lengthscales缩放；σ²是方差
    # gpflow.kernels.Product 以乘法的方式将核组合起来（应该是为了针对输入点为高维的情况）
    kernel = gpflow.kernels.Product([   gpflow.kernels.RBF(lengthscale=lengthscale, active_dims=[i, i + input_dims]) 
                                     for i in range(input_dims)   ])

    # 构建要拟合的GP模型：Scalable Variational Gaussian Process Classification(SVGP) (Hensman, et. al. 2013) 
    # 核函数：上述的核函数
    # 似然：伯努利似然，即二项分布；link function是tf.math.sigmoid
    # 变量（数据集）：X
    # whiten is a boolean. If True, we use the whitened representation of the inducing points.
    m = gpflow.models.SVGP(kernel=kernel,
                           likelihood=gpflow.likelihoods.Bernoulli(invlink=tf.math.sigmoid),
                           inducing_variable=X,
                           whiten=False)

    # 控制可训练的变量
    m.inducing_variable.Z.trainable = False

    # 优化器的选择：tensorflow中的RMSprop
    optimizer = tf.keras.optimizers.RMSprop(rho=0.0)

    ## 定义计算误差的lambda表达式为loss()
    # log_likelihood(X, Y)：This gives a variational bound on the model likelihood.
    loss = lambda: -m.log_likelihood(X, y) + lengthscale_regularizer(m.kernel)
    
    # 计算当前的loss
    prev_loss = loss().numpy()

    # 开始优化的过程，优化的最大次数为num_steps
    for i in range(num_steps):
        # 优化的目标是：优化trainable_variables，使loss最小
        optimizer.minimize(loss, m.trainable_variables)
        # 计算本次优化的loss（这里是为了方便下面的终止条件判断）
        current_loss = loss().numpy()
        
        if i % 500 == 0:          # 每500轮输出一次信息
            print('Loss at step %s: %s' % (i, current_loss))
        
        # 若两次迭代间的误差变化很小，则终止优化的过程
        if abs((current_loss - prev_loss) / prev_loss) < 1e-7:
            print('Loss at step %s: %s' % (i, current_loss))
            break
        # 更新loss，准备下次迭代
        prev_loss = current_loss

    # 调用可视化模型函数，将本轮优化的结果展示为图片
    visualize_model(X, y, m, title=title)
    
    return m

## 定义logistic函数
def logistic(x):
    return 1 / (1 + tf.math.exp(-x))

## 计算logistic函数的平方
def logistic_square(x):
    return tf.math.square(logistic(x))

## 计算方差积分
def variance_integral(means, variances):
    # 方法说明：Computes N Gaussian expectation integrals of a single function ‘f’ using Gauss-Hermite quadrature. 
    # 参数：H为Number of Gauss-Hermite evaluation points；Din为Number of input dimensions.
    return gpflow.quadrature.mvnquad(logistic_square, means, variances, H=50, Din=1)

## 计算期望积分
def expected_integral(means, variances):
    return gpflow.quadrature.mvnquad(logistic, means, variances, H=50, Din=1)

## 计算模型高斯过程的方差
def variance_logistic_f(m, x):
    # gpflow.models.SVGP.predict_f(x)能够计算模型输入数据x的均值和方差
    means, variances = m.predict_f(x)
    variances = tf.expand_dims(variances, axis=1)    # 扩大维度，扩大张量的维度索引为1
    expected_logistic_squared = tf.math.square(expected_integral(means, variances))
    return variance_integral(means, variances) - expected_logistic_squared

## 随机傅里叶特征方法采样的实现
# 给出模型的W和b，根据X采样出模型对应的值

def fourier_features(X, W, b):
    """
    Given sampled tensors W and b, construct Fourier features of X. We do not take into account the normalizing constant
    alpha in this method as it is only used for optimization, and the constant does not affect the location of the
    maximizing point.
    :param X: tensor of shape (count, n, d)
    :param W: tensor of shape (count, D, d)
    :param b: tensor of shape (count, D, 1)
    :param kernel: gpflow kernel
    :return: tensor of shape (count, n, D)
    """
    D = W.shape[1]

    # tf.linalg.matrix_transpose(X)：Transposes last two dimensions of tensor X
    WX_b = W @ tf.linalg.matrix_transpose(X) + b  # (count, D, n)
    # RFF的计算公式即下式
    return tf.cast(tf.sqrt(2.0 / D), dtype=tf.float64) * tf.cos(tf.linalg.matrix_transpose(WX_b))  # (count, n, D)

## 根据模型的kernel计算出模型的傅里叶采样特征W和b
# 返回输入X对应的查询值，以及模型特征W和b
# X应包括整个样本空间（要据此计算出count和d，如这里的X的长度应该是1000*1000）
# D为RFF中傅里叶特征的数量，默认为100

def sample_fourier_features(X, kernel, D=100):
    """
    Fourier feature mapping of X for the RBF kernel
    :param X: tensor of shape (count, n, d)
    :return: tensor of shape (count, n, D)
    """
    count = X.shape[0]           # 给出的样本点的数量
    d = X.shape[2]               # 样本点的维度

    if type(kernel) == gpflow.kernels.base.Product:  # For DTS implementation where we use a product of kernels
        # 如果kernel是通过product方法得到的，则按照下面的方法计算lengthscales
        input_dims = d // 2
        lengthscales = np.zeros(d)
        for i in range(input_dims):
            k = kernel.kernels[i]
            lengthscale_val = k.lengthscale.numpy()
            lengthscales[i] = lengthscale_val
            lengthscales[i + input_dims] = lengthscale_val
    else:
        lengthscales = kernel.lengthscale.numpy()

    # 计算模型的W和b
    W = tf.random.normal(shape=(count, D, d), mean=0.0, stddev=1.0 / lengthscales,dtype=tf.float64)
    b = tf.random.uniform(shape=(count, D, 1), minval=0, maxval=2 * np.pi, dtype=tf.float64)

    # 返回输入X对应的查询值，以及模型特征W和b
    return fourier_features(X, W, b), W, b  # (count, n, D)

## 采样得到theta
# 采样公式：q(theta|D) = /int p(theta|y)p(y|f)q(f|D) df dy

def sample_theta_variational(phi, q_mu, q_sqrt):
    """
    Samples from distribution q(theta|D) = /int p(theta|y)p(y|f)q(f|D) df dy
    :param phi: Fourier features tensor with shape (count, n, D)
    :param q_mu: tensor of shape (n, 1)
    :param q_sqrt: tensor of shape (1, n, n). Lower triangular matrix
    :return: tensor with shape (count, D, 1)
    """
    count = phi.shape[0]
    n = phi.shape[1]
    D = phi.shape[2]

    standard_normal = tfp.distributions.MultivariateNormalDiag(
            loc=tf.zeros(n, dtype=tf.float64),
            scale_diag=tf.ones(n, dtype=tf.float64))
    standard_normal_samples = standard_normal.sample(count)
    # (count, n)

    q_samples = q_sqrt @ tf.expand_dims(standard_normal_samples, axis=-1) + q_mu
    # (count,n,1)

    transposed_phi = tf.linalg.matrix_transpose(phi)
    # (count, D, n)

    transform_mat = tf.cond(n >= D,
        true_fn = lambda : tf.linalg.inv(transposed_phi @ phi) @ transposed_phi,
        false_fn = lambda: transposed_phi @ tf.linalg.inv(phi @ transposed_phi) ) # (count, D, n)

    theta = transform_mat @ q_samples
    # (count, D, 1)

    return theta

4.5.2 采样函数的定义¶

这里的采样是指从GP上采样，而不是从目标函数上采样。

In [40]:
## 采样函数
# 根据模型m查询query_points的值，combs是duels的样本空间，有1000*1000种情况
# D为RFF中傅里叶特征的数量，默认为1000

def sample_f(m, query_points, combs, D=1000):
    """
    Generates a sample f using continuous Thompson sampling.
    :param m: gpflow model
    :param query_points: Input points corresponding to trained gpflow model. Tensor of shape (n, d)
    :param combs: tensor of shape (num_discrete_points ** 2, input_dims * 2). all combinations of discrete points
    :return: tensor of shape (num_discrete_points ** 2, 1)
    """
    # 这里的X已经不是每一轮的测试点，而是整个样本空间X
    X = np.expand_dims(combs, axis=0)

    # 根据GP模型的核函数计算出模型的傅里叶特征W和b，并返回所有X（整个样本空间）对应的GP观测值
    # 全部样本的观测值用phi存储，实际上phi即此刻GP给出的模拟值
    phi, W, b = sample_fourier_features(X, m.kernel, D)
    
    # 根据GP模型的傅里叶特征W和b，查询待查寻点query_points对应的值，记为phi_y
    phi_y = fourier_features(tf.expand_dims(query_points, axis=0), W, b)
    # 这里的m.q_mu和m.q_sqrt对应GPmodel的变分特性，由此计算出theta的值
    theta = sample_theta_variational(phi_y, m.q_mu, m.q_sqrt)
    
    # tf.squeeze删除维度为1的维度，给出最终的采样结果
    return tf.squeeze(phi @ theta, axis=0)

4.5.4 找出下一对duel（迭代step.2）¶

定义Soft-Copeland函数

In [41]:
## 找出Condorcet winner的函数（即确定下一个duel的x点）
# 根据论文公式计算的

def soft_copeland_maximizer(f_vals, discrete_space):
    """
    Given function evaluations, calculate the Condorcet winner.
    :param f_vals: tensor of shape (num_discrete_points ** 2, 1)
    :param discrete_space: tensor of shape (num_discrete_points, input_dims). all discrete points
    """
    num_discrete_points = int(np.sqrt(f_vals.shape[0]))   # f_vals的shape是(num_discrete_points ** 2, 1)
    # 下面这句参见论文里的公式
    soft_copeland = np.mean(np.reshape(logistic(f_vals),
                                       [num_discrete_points, num_discrete_points]), axis=1)  # (num_discrete_points)
    return discrete_space[np.argmax(soft_copeland)]
